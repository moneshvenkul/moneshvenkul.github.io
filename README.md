# Monesh Venkul Vommi – Portfolio Website

A personal portfolio site built with React to showcase my experience, skills, projects, and publications.

## Live Site

- https://moneshvenkul.github.io/

## Highlights

- Professional profile focused on full-stack engineering and cloud-native development.
- Experience timeline across enterprise, education, and startup environments.
- Skills coverage for frontend, backend, data, and cloud technologies.
- Contact section with LinkedIn, GitHub, and email.

## Tech Stack

- React
- JavaScript (ES6)
- CSS3
- GitHub Pages

## Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/moneshvenkul/moneshvenkul.github.io.git
   cd moneshvenkul.github.io
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm start
   ```
4. Open http://localhost:3000

## Build

```bash
npm run build
```

## Deploy

This project is configured for GitHub Pages deployment.

## Customize Content

Most profile content is maintained in:

- `src/portfolio.js`

Update this file to edit:

- Intro and summary
- Social links
- Skills and tools
- Education and certifications
- Work experience
- Contact details
