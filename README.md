## Portfolio Site — Local Setup

```bash
# Clone
git clone https://github.com/moneshvenkul/moneshvenkul.github.io.git
cd moneshvenkul.github.io

# Install
npm install

# Development server (http://localhost:3000)
npm start

# Production build
npm run build

# Deploy to GitHub Pages
npm run deploy
```

All portfolio content is configured in [`src/portfolio.js`](src/portfolio.js).
Theme tokens live in [`src/theme.js`](src/theme.js).
The built-in AI chatbot knowledge base is at [`src/data/chatbotKnowledge.js`](src/data/chatbotKnowledge.js).
